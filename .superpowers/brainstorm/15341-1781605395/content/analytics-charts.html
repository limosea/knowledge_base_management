Analytics页面详细图表

Analytics页面应展示哪些深度分析图表？

📊

知识分类分布图

饼图或柱状图，展示各分类的知识条目数量占比

数据源: byCategory

📈

质量分数分布

展示知识条目的质量分数分布情况（平均分、最高分、最低分）

数据源: avgQualityScore

🔑

API Key状态统计

饼图展示活跃/过期/禁用的API Key分布

数据源: apiKeys.total/active/expired

📉

请求量趋势图

折线图展示今日/本周/本月的请求量变化趋势

数据源: requests.today/thisWeek/thisMonth

📝

操作类型统计

柱状图展示各类操作（登录、创建、更新、删除）的发生频率

数据源: 审计日志统计

🕐

操作时间线

时间轴展示系统关键操作事件

数据源: 审计日志

💡 推荐全选，Analytics作为深度分析页面应提供全面视角
